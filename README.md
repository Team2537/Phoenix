# Phoenix
Team 2537 2018 FIRST Power Up Robot Code  
## Joystick Buttons:
### Left Joystick:  
1. Intake cube
2. Lower vertical slider
3. Raise vertical slider
4. Flip cube manipulator down
5. Flip cube manipulator up  
### Right Joystick:  
1. Expel cube
2. Stop climb
3. Start climb
4. Close ramp latch
5. Deploy ramp
